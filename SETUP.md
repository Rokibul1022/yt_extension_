# Setup Instructions

## 1. Start the Backend

```bash
cd c:\Users\User\Desktop\yt-downlaoder-ext
python server.py
```

Server runs on http://localhost:5000

## 2. Install Chrome Extension

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `extension` folder: `c:\Users\User\Desktop\yt-downlaoder-ext\extension`
5. Extension icon appears in toolbar

## 3. Use It

1. Click the extension icon
2. Paste YouTube URL
3. Select MP4 or MP3
4. Click Download
5. File saves to your Downloads folder

Note: Download a proper icon.png (128x128) and replace the placeholder in the extension folder.
